# 三角函数 — 高中数学核心知识点

## 基本概念

### 正弦（sin）
在直角三角形中，锐角A的对边与斜边的比值称为正弦，记作 sinA = 对边/斜边。
单位圆定义：对于任意角θ，其终边与单位圆的交点的纵坐标即为sinθ。

### 余弦（cos）
锐角A的邻边与斜边的比值，记作 cosA = 邻边/斜边。
单位圆定义：交点的横坐标即为cosθ。

### 正切（tan）
锐角A的对边与邻边的比值，记作 tanA = 对边/邻边 = sinA/cosA。

## 诱导公式（口诀：奇变偶不变，符号看象限）
- sin(π/2 - θ) = cosθ
- cos(π/2 - θ) = sinθ
- sin(π + θ) = -sinθ
- cos(π + θ) = -cosθ
- tan(π + θ) = tanθ

## 和差公式
- sin(α+β) = sinαcosβ + cosαsinβ
- sin(α-β) = sinαcosβ - cosαsinβ
- cos(α+β) = cosαcosβ - sinαsinβ
- cos(α-β) = cosαcosβ + sinαsinβ
- tan(α+β) = (tanα+tanβ)/(1-tanαtanβ)

## 二倍角公式
- sin2θ = 2sinθcosθ
- cos2θ = cos²θ - sin²θ = 2cos²θ - 1 = 1 - 2sin²θ
- tan2θ = 2tanθ/(1-tan²θ)

## 学习重点
1. 单位圆是理解三角函数的几何基础
2. 诱导公式不必死记，掌握"奇变偶不变"口诀即可
3. 和差公式与二倍角公式是高考高频考点

## 常见错误
- 混淆 sin(α+β) 和 sinα+sinβ（它们不相等）
- 诱导公式符号弄错
- 倍角公式中 cos2θ 三种形式不会灵活切换
